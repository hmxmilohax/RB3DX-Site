How to Install

Rock Band 3 Deluxe requires a copy of vanilla Rock Band 3 in order to run. We hope this is clear.

  • PS3: BLUS30463 (USA)
  • 360: 45410914 (World)
  • Wii: SZBE69 (USA) or SZBP69 (EU)

Select your platform of choice

Installing on PS3*

*RB3DX requires a modded PS3 console (CFW or HEN) in order to run.

Download Rock Band 3 Deluxe if you haven't already

  • Extract the .zip file.
  • Copy the .pkg file to the root of a FAT32 formatted USB drive.
  • Remove it from your PC and plug it into the far right USB port on your PS3.
  • On the XMB, navigate to Package Manager > Install Package Files > Standard.
  • Install it just like any other package.

✅ Rock Band 3 Deluxe is now installed and ready to play!

You can play the game using the original disc or via a backup on your hard drive. Rock on!

Installing on Xbox 360*

*RB3DX requires a modded Xbox 360 console (RGH/JTAG) in order to run.

*Games on Demand (GoD) or ISO disc images are NOT supported!

*The game must be in extracted folder format. You should see a `default.xex` and `GEN` folder.

Install your copy of Rock Band 3 to the console's hard drive. This step is required.

  • We recommend that you rename its default.xex to default_vanilla.xex, just as a backup in case you ever want to go back to vanilla (for whatever reason).

Download Rock Band 3 Deluxe if you haven't already

  • Extract the .zip file.
  • Copy the contents of it to where you have Rock Band 3 installed.
    • You can do this via FTP or by putting the files on a USB drive and copying them using Aurora's file manager.
  • Select Yes to overwrite the files if it asks you to.

Before you launch the game for the first time, we recommend that you...

  • Disable any title updates you may have for Rock Band 3 in Aurora.
    • Rock Band 3 Deluxe rolls the latest one into its installation.
    • If you've never played Rock Band 3 on your console, you don't need to do this.
  • Clear your system cache.
    • In the dashboard, navigate to System Settings > Storage and press Y to clear the system cache.
    • If you've never played Rock Band 3 on your console, you don't need to do this.
  • Download the Rock Band 3 Deluxe cover art in Aurora.
    • Hover over Rock Band 3 in your games list and press Y.
    • Scroll all the way down, then scroll left until you get to Download Cover.
    • Download the one uploaded by jnack, it should be all the way at the bottom.
    • You can also rename the game in the list to Rock Band 3 Deluxe if you so choose.

✅ Rock Band 3 Deluxe is now installed and ready to play!

PS, you should also check out RB3Enhanced! You can run it alongside Deluxe and use its additional quality of life features. Rock on!

Installing on Wii* (and Dolphin Emulator)

*RB3DX requires a modded (Homebrewed) Wii console in order to run.

*You can run RB3DX on Wii, vWii, Wii Mini, and Dolphin Emulator.

Download Rock Band 3 Deluxe if you haven't already

  • Extract the .zip file.
  • Drag in a USA or PAL copy of vanilla Rock Band 3 (.iso or .wbfs format) into the extracted folder.
  • Run patch_images.bat if you're on Windows or patch_images.sh if you're on Linux
    • This may take a while, press any key to continue when prompted.
  • You should see a new folder called rock-band-3-deluxe. Inside it, you will find your newly created .iso or .wbfs file.
  • On a real Wii:
    • Copy this file to the wbfs folder on your USB drive to load with a USB loader. We recommend USB Loader GX.
  • On Dolphin Emulator:
    • Copy this file to where you have Dolphin set to scan for your games.

✅ Rock Band 3 Deluxe is now ready to play!

PS, you should also check out RB3Enhanced! You can run it alongside Deluxe and use its additional quality of life features. Rock on!

Installing song updates for YARG

RB3DX song updates require original Rock Band CON files to work.

Download YARG Song Updates from Misc. Goodies if you haven't already

  • Extract the .zip file.
  • Drag the songs_updates folder next to the folder that contains your Rock Band CON files.
  • Do not drag songs_updates into the folder with your Rock Band songs
  • In YARG, navigate to Settings > Songs > Scan Songs.

✅ Rock Band 3 Deluxe Song Updates are now ready to play!
