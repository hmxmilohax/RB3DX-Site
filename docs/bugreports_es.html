Reportes de Bugs

Manejamos reportes de bugs y rastreamos problemas a través de GitHub, si reportas bugs en nuestro Discord o lo envías a nosotros directamente, probablemente no lo vimos. La mejor manera de avisar sobre un problema es crear un problema (Issue) en el repositorio de GitHub.

Los problemas actuales se pueden encontrar aquí. Si tienes una cuenta de GitHub, aquí puedes levantar un problema siguiendo la plantilla.
