Decompilación de Rock Band 3

¿Qué es esto?

Esto es un proyecto de ingeniería inversa que va a crear una decompilación que produce el código exacto de la versión Wii de Rock Band 3. Este proceso involucra programación de código C++ que, cuando sea compilada, produce el mismo código que la versión oficial del juego.

Esto va a resultar en una versión del juego que puede ser editada con código C++ en lenguaje claro. Esto nos va a dejar hacer modificaciones más avanzadas de lo que es posible con Rock Band 3 Deluxe.

¿Por qué usan la versión de Wii?

Simplemente, es la única versión del juego con la que este tipo de proyecto es posible. Las otras versión del juego pasaron por optimización de tiempo de enlace que hace el rendimiento del código más rápido, pero ofusca el código completamente. Esto aumenta la dificultad exponencialmente.

¿Puedo contribuir?

Puedes contribuir aquí. Siempre se puede usar más ayuda para avanzar este proyecto.

¿Por qué mejor no hacen una versión para PC?

En primer lugar, los proyectos de decompilación no son para hacer versiones para otras plataformas. Dicho esto, esta decompilación es un caso especial donde el proyecto también quiere crear una versión para PC y otras plataformas.

Por favor, toma en cuenta que este proyecto va tomar mucho tiempo y no va ver ningún intento a crear una versión de PC hasta que el proyecto de decompilación termine. Si quieres que avance más rápido y sabes como ayudar (o quieres aprender), considera contribuir.

¿Se va poder hacer [modificación] con esto?

En teoría si, pero las modificaciones no son una prioridad para la decompilación. La decompilación está atinando a crear una versión exacta con código C++ en lenguaje claro. Una vez realizada, se pueden crear modificaciones.
